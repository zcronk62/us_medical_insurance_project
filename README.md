# U.S. Medical Insurance Cost Portfolio Project

## Description of Project:

This project provides some descriptive analysis for a simulated U.S. medical insurance cost dataset provided by Codecademy.
The purpose of this project was to determine average values for age, BMI and annual cost for clients represented within the
data. These average values were then compared amongst males and females, smokers and non-smokers, which regions the clients
resided in and the number of children they had. 

I know that the analysis I performed in the project can be complete in a tidier fashion using the `numpy` and `pandas` Python
libraries. I used dictionaries and list to showcase my ablitiies working with these object types.

## Features

Within the jupyter notebook, I created several functions for analysis. These functions create and act on dictionaries to perform
the desired analysis.
